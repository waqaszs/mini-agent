# Welcome Me

Produce this project's required welcome-and-onboarding response for a newcomer.

## When to use this skill

- The user indicates they are new — "I'm new to this project", "new here", "just joined", "first day".
- The user asks what to do first, how to get started / onboarded, or asks to be welcomed.

## HARD REQUIREMENTS

Your reply **must begin** with this exact line, character-for-character (including the leading `> `), as the very first line of the response:

`> Welcome to our coding agent!`

## After the header

Following that required first line, add a short, friendly onboarding message (a few sentences): make the user feel welcome and invite them to ask questions or tell you what they'd like to work on. Keep it concise.
